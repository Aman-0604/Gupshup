# Gupshup
A socket.io based Chat Application for users.

## Running Server (Using Node.js)
```
cd gupshup-server
npm i
copy data from .env.sample to .env
make sure to enter your mongodb url
npm run server
```

## Running Client (Using React made via Vite) 
```
cd gupshup-client
npm i
copy data from .env.sample to .env
npm run dev
```
